# 骚神Chrome浏览器VIP插件
![alt](/img/haibao.png)
| 特性分类 | 具体内容 |
| ---- | ---- |
| **支持系统** | Windows、Mac、Linux 三大PC系统 |
| **支持浏览器** | Chrome谷歌浏览器、Edge微软浏览器、360极速浏览器、QQ浏览器、360浏览器等所有Chrome内核浏览器 |
| **支持定位元素语法** | CSS、XPath、坐标 |
| **扩展定位能力** | 支持定位iframe内元素的语法 |
| **代码生成支持** | 可生成 DrissionPage、Playwright、Selenium 定位代码 |
| **定位操作场景** | 1. 网页内直接定位 2. DevTools 工具内定位 3. 右键菜单快捷定位 |
| **应用场景** | 1. 自动化测试 2. 爬虫 3. 前端开发 4.JS逆向 |
